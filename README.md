# StableDEX

Multi-network perpetuals DEX for trading USDC/EURC pairs with leverage.

## Features
- ✅ Multi-network support (ETH Sepolia, Arc Testnet, Base Sepolia)
- ✅ EVM wallet integration
- ✅ Real-time price charts
- ✅ Long/Short perpetual positions
- ✅ PNL tracking (USDC & percentage)
- ✅ Variable leverage (1x-10x)

## Setup
```bash
npm install
npm start
```

## Networks
- ETH Sepolia
- Arc Testnet
- Base Sepolia
